* [문제 링크](https://www.acmicpc.net/problem/16235)   

* 풀이 방법  
  - 특별한 알고리즘을 요하지는 않지만, 그냥 풀면 시간초과가 나는 문제입니다.  
  
  
## 내 코드  

```python
import sys
input = sys.stdin.readline


def solve():
    n, m, k = map(int, input().split())
    land = [[5 for _ in range(n)] for _ in range(n)]
    #나무 정보를 저장할 dict로 이루어진 배열
    tree = [[{} for _ in range(n)] for _ in range(n)]
    S2D2 = [list(map(int, input().split())) for _ in range(n)]
    for _ in range(m):
        r, c, a = map(int, input().split())
        tree[r-1][c-1][a] = 1
    for _ in range(k):
        ng = [[{} for _ in range(n)] for _ in range(n)]
        for r in range(n):
            for c in range(n):
                #나무가 없다면 양분만 추가해줌
                if not tree[r][c]:
                    land[r][c] += S2D2[r][c]
                    continue
                tmp = 0
                more = 0
                #dict에 저장된 값들을 나이를 기준으로 정렬
                for age, count in sorted(tree[r][c].items()):
                    #자랄 수 있는 나무의 수를 구함
                    can = min(land[r][c]//age, count)
                    #자랄 수 있는 나무가 있으면
                    if can:
                        #그만큼 양분을 빼주고
                        land[r][c] -= age*can
                        if age+1 not in ng[r][c]:
                            ng[r][c][age+1] = 0
                        #다음 세대에 표시해준다
                        ng[r][c][age+1] += can
                        #(age+1)이 5의 배수이면 나무는 가을에 변식을 할 수 있다.
                        if (age+1) % 5 == 0:
                            tmp += can
                    #죽은 나무에서 나온 양분
                    more += age//2*(count-can)
                land[r][c] += more + S2D2[r][c]
                #번식할 수 있는 나무가 있다면 검사
                if tmp:
                    if r > 0:
                        if 1 not in ng[r-1][c]:
                            ng[r-1][c][1] = 0
                        ng[r-1][c][1] += tmp
                        if c > 0:
                            if 1 not in ng[r-1][c-1]:
                                ng[r-1][c-1][1] = 0
                            ng[r-1][c-1][1] += tmp
                        if c < n-1:
                            if 1 not in ng[r-1][c+1]:
                                ng[r-1][c+1][1] = 0
                            ng[r-1][c+1][1] += tmp
                    if r < n-1:
                        if 1 not in ng[r+1][c]:
                            ng[r+1][c][1] = 0
                        ng[r+1][c][1] += tmp
                        if c > 0:
                            if 1 not in ng[r+1][c-1]:
                                ng[r+1][c-1][1] = 0
                            ng[r+1][c-1][1] += tmp
                        if c < n-1:
                            if 1 not in ng[r+1][c+1]:
                                ng[r+1][c+1][1] = 0
                            ng[r+1][c+1][1] += tmp
                    if c > 0:
                        if 1 not in ng[r][c-1]:
                            ng[r][c-1][1] = 0
                        ng[r][c-1][1] += tmp
                    if c < n-1:
                        if 1 not in ng[r][c+1]:
                            ng[r][c+1][1] = 0
                        ng[r][c+1][1] += tmp
        tree = ng
    print(sum(sum(tree[i//n][i % n].values()) for i in range(n**2)))


if __name__ == '__main__':
    solve()
```
  
별다른 알고리즘이 필요하지 않지만 자료구조를 잘 선택하고, 적절하게 최적화해야 시간 안에 들어올 수 있는 단순 구현문제였습니다.
